# cppseed

`cppseed` is a dependency-free C++ CLI for generating small, buildable CMake projects.

> Status: early design phase

Design documents:

- [v0.1 requirements](docs/requirements.md)
- [v0.1 external design](docs/basic-design.md)
- [v0.1 internal design](docs/detailed-design.md)
- [v0.1 test plan](docs/test-plan.md)

## Planned usage

```console
cppseed new my-project
```

The first release will generate a C++ project that can be built and tested with CMake and CTest.
