# Architecture Decision Records

複数initiativeへ影響する設計判断をこのディレクトリに記録する。
initiative固有の判断は `docs/initiatives/<initiative>/adr/` に置く。

- file name: `NNNN-short-decision-title.md`
- status: Proposed、Accepted、Superseded、Rejectedのいずれか
- accepted ADRは内容を書き換えず、新しいADRからsupersedeする
- template: [ADR template](../templates/adr-template.md)
